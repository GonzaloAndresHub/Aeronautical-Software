# ✈️ SkyFox — Aeronautical Software

> Internal aeronautical management platform for flight operations, technical documentation, and personnel administration.

---

## 📋 Overview

**SkyFox** is a desktop application designed for internal use within aeronautical companies. It provides an integrated environment to manage flight operations, visualize technical documents, and administer aeronautical personnel — all from a single platform.

---

## 🚀 Features

- 📊 **Operations Dashboard** — Real-time monitoring of aeronautical operations and key indicators
- 📁 **Technical Document Viewer** — Access and visualization of technical manuals, PDFs, and aeronautical reports
- 👥 **Personnel Management** — Administration of aeronautical staff including pilots, engineers, and ground crew
- 🌙 **Dark / Light Mode** — Adaptive interface for different work environments

---

## 📦 Installation
```bash
# Clone the repository
git clone https://github.com/GonzaloAndresHub/Aeronautical-Software.git

# Navigate to the project folder
cd Aeronautical-Software/WebSite\ SkyFox

# Install dependencies
npm install

# Run the application
npm start
```
